== Description ==

Youtube Broken Link Checker Hide all links to bad youtube videos to improve SEO.

Youtube Broken Link Checker monitors and checks all internal and external links on your site for broken youtube links. Hide all links to bad youtube videos to improve SEO and user experience.
Once installed, the plug-in, while viewing a post, will start analyzing the content and looking for any broken YouTube video links. Once the analysis is done, the plugin will start checking each link to see if it works. 
Broken links, if detected, will be hidden. This is to prevent search engines from penalizing your content. The plugin will send an email to the administrator to notify the operation.

== Installation ==

The plugin is simple to install:

1. Download `youtube-broken-link-checker.zip`
2. Unzip
3. Upload `youtube-broken-link-checker` directory to your `/wp-content/plugins` directory, or install the plugin throught the Wordpress plugins screen directly.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Go to *Settings > youtube-broken-link-checker to configure the plugin.
