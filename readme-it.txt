Auto Replace Broken Links For Youtube sostituisce automaticamente tutti i link relativi ai video di YouTube, interrotti o non più validi, nei tuoi post per migliorare la SEO e garantire sempre la tua autorevolezza sui motori di ricerca. Tutta l'attività del Plugin è eseguita in modalità asincrona.

== Descrizione ==

Auto Replace Broken Links For Youtube monitora e controlla tutti i collegamenti a video di Youtube, sul tuo sito web, interrotti o non più raggiungibili. Sostituisce, automaticamente, tutti i link video di YouTube ripristinandoli per migliorare la SEO e l'esperienza dell'utente.
Una volta installato, il plugin, durante la visualizzazione di un post, inizierà ad analizzare il contenuto e a cercare eventuali collegamenti video di YouTube interrotti. Una volta terminata l'analisi, il plugin, inizierà a controllare ogni collegamento per verificarne la validità.
I collegamenti interrotti, se rilevati, verranno sostituiti con altri link di youtube, cercati autonomamente, in base alla frase chiave del post. Questo per evitare che i motori di ricerca penalizzino i tuoi contenuti. Il plugin invierà un'e-mail all'amministratore per notificare l'esito dell'operazione. Tutta l'attività del Plugin è eseguita in modalità asincrona.

== Installazione ==

Il plugin è semplice da installare:

1. Scarica 'auto-replace-broken-links-for-youtube.zip'
2. Decomprimi il file zip.
3. Carica la directory 'auto-replace-broken-links-for-youtube', decompressa, nella tua directory '/wp-content/plugins' o installa il plug-in direttamente dalla schermata dei plug-in di Wordpress.
4. Attiva il plug-in tramite il menu "Plugin" presente nella dashboard di WordPress.
5. Vai su *Impostazioni > Auto Replace Broken Links For Youtube e configura il plugin inserendo la tua Youtube API Key.
